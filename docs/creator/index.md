# Creator — 桌面 AI 内容工厂

## 什么是 Creator？

**Lurus Creator** 是一个桌面端 AI 内容创作工具，将视频下载、音频转写、AI 改写、多平台发布整合为一条自动化流水线。单个 `.exe` 文件，零外部依赖，打开即用。

> 当前版本：v0.3.0（Beta）

---

## 核心能力

### 一键内容流水线

```
视频来源               处理管道              发布目标
┌──────────┐    ┌────────────────┐    ┌──────────────┐
│ YouTube  │    │  yt-dlp 下载   │    │  微信公众号  │
│ Bilibili │ →  │  ffmpeg 转码   │ →  │  抖音        │
│ 本地文件 │    │  Whisper 转写  │    │  小红书      │
└──────────┘    │  LLM 改写/翻译 │    └──────────────┘
                └────────────────┘
```

从原始视频到多平台发布，全程自动化：

| 阶段 | 工具 | 说明 |
|------|------|------|
| **下载** | yt-dlp | 支持 YouTube、Bilibili 等 1000+ 视频平台 |
| **转码** | ffmpeg | 视频格式转换、裁剪、水印去除 |
| **转写** | Whisper | 语音转文字，支持 99 种语言 |
| **改写** | LLM (Lurus API) | AI 润色、翻译、格式调整 |
| **发布** | chromedp | 无头浏览器自动登录、上传、发布 |

### 智能改写

不只是简单搬运，Creator 用 AI 对内容进行深度加工：

- **翻译**: 英文视频 → 中文文案（保留专业术语）
- **改写**: 调整语气风格（严肃→轻松、长文→短文）
- **提取**: 从长视频中提取关键观点，生成短文案
- **SEO 优化**: 自动生成标题、标签、摘要

### 多平台一键发布

通过 chromedp 无头浏览器技术，自动完成各平台的登录和发布：

| 平台 | 支持的内容类型 |
|------|--------------|
| **微信公众号** | 图文文章 |
| **抖音** | 短视频 + 文案 |
| **小红书** | 图文笔记 |

::: tip
首次使用各平台发布功能时，需要手动扫码登录一次。之后 Creator 会保存登录状态。
:::

### DevFactory 模式

内置的任务管理引擎，支持批量内容生产：

- **SQLite 状态机**: 每个任务有独立状态追踪（下载→转写→改写→发布）
- **批量处理**: 导入视频 URL 列表，自动排队处理
- **预算控制**: 设定 LLM Token 使用上限，避免超支
- **断点续传**: 任务中断后从上次完成的阶段继续

## 技术架构

```
┌─────────────────────────────────────┐
│          Creator Desktop            │
│     Wails v2 (Go + TypeScript)      │
│                                     │
│  ┌───────────┐  ┌───────────────┐  │
│  │  前端 UI   │  │   Go 后端     │  │
│  │ TypeScript │  │              │  │
│  │            │  │ ┌──────────┐ │  │
│  │ 任务面板   │  │ │ yt-dlp   │ │  │
│  │ 编辑器    │  │ │ ffmpeg   │ │  │
│  │ 设置      │  │ │ Whisper  │ │  │
│  │            │  │ │ chromedp │ │  │
│  └───────────┘  │ └──────────┘ │  │
│                  │              │  │
│                  │ ┌──────────┐ │  │
│                  │ │ SQLite   │ │  │
│                  │ │ 状态机   │ │  │
│                  │ └──────────┘ │  │
│                  └──────────────┘  │
└─────────────────────────────────────┘
         │
         ▼
    Lurus API (LLM)
```

- **前端**: TypeScript，提供任务管理和内容编辑界面
- **后端**: Go，调度下载/转码/转写/发布管道
- **桌面框架**: Wails v2，编译为单个可执行文件
- **LLM 接入**: 通过 [Lurus API](/guide/introduction) 使用 DeepSeek、GPT 等模型
