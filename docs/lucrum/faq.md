# 常见问题

## 账号与入门

### Q: 如何注册 Lucrum？

访问 [lucrum.lurus.cn](https://lucrum.lurus.cn)，使用 Lurus 统一账号登录。如果还没有 Lurus 账号，注册一个即可——所有 Lurus 产品共享同一个账号。

### Q: 需要付费吗？

Lucrum 提供免费和付费两种使用方式：

| 功能 | 免费 | 付费 |
|------|------|------|
| AI 交易助手 | 每日有限对话次数 | 无限制 |
| 策略市场浏览 | 全部可见 | 全部可见 |
| 免费策略 | 可使用 | 可使用 |
| 付费策略订阅 | 不可用 | 可订阅 |
| 策略开发 | 基础回测 | 完整功能 |

超出免费额度后，从 [鹿贝钱包](/platform/billing#wallet) 扣费。

### Q: 支持哪些市场？

当前支持 A 股（沪深两市）。港股、美股、加密货币正在接入中。

## AI 助手

### Q: AI 助手的分析准确吗？

AI 助手基于大语言模型和技术分析工具提供参考意见，**不构成投资建议**。它擅长：
- 快速解读技术指标和 K 线形态
- 梳理市场事件和逻辑
- 辅助策略构思和代码编写

它不擅长预测短期涨跌。请将 AI 分析作为决策参考，而非依据。

### Q: AI 助手会记住我之前的对话吗？

会。Lucrum 集成了 [MemX 记忆引擎](/memx/)，AI 助手能记住你的交易偏好、关注的板块和历史对话。使用越多，它越懂你。记忆按用户隔离，不会泄露给其他用户。

### Q: 可以用 AI 助手写策略代码吗？

可以。描述你的策略思路，AI 会生成 Python 代码框架。例如：

```
你: 帮我写一个 RSI 超卖反弹策略，RSI 低于 30 买入，高于 70 卖出

AI: 好的，以下是策略代码...（生成完整可运行的代码）
```

生成的代码可以直接在策略工作台中回测验证。

## 策略相关

### Q: 策略的回测数据可信吗？

回测使用真实历史行情数据，但需注意：
- 回测不考虑冲击成本和滑点（大资金可能有差异）
- 过度优化参数可能导致"过拟合"
- 过去的表现不能保证未来收益

建议回测后先用模拟盘验证一段时间。

### Q: 如何提现策略收入？

策略订阅收入以鹿贝形式进入你的钱包。提现路径：
1. 登录 [admin.lurus.cn](https://admin.lurus.cn) → 钱包管理
2. 点击「提现」
3. 输入金额和银行卡信息
4. 通常 1-3 个工作日到账

### Q: 策略代码会被泄露吗？

不会。策略代码在服务端加密存储，使用者只能看到策略的描述、指标和回测报告，无法查看源代码。

## 技术问题

### Q: API 请求返回 429？

超出了请求频率限制。Lucrum API 对不同套餐有不同的速率限制，降低请求频率或升级套餐。

### Q: 策略执行延迟很高？

检查：
- 网络连接是否稳定
- 策略逻辑是否有复杂计算（建议预计算缓存）
- 当前市场时段是否为高并发（如开盘/收盘）

### Q: 支持哪些编程语言编写策略？

当前支持 Python。Lucrum 策略 SDK 提供了丰富的技术指标库和交易执行接口。
