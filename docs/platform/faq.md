# 常见问题

## 账号

### Q: 一个账号可以使用所有 Lurus 产品吗？

是的。Lurus 采用统一账号体系，注册一次即可登录所有产品（Lurus API、Lucrum、Switch、Creator 等），共享同一个钱包余额和订阅计划。

### Q: 如何修改密码？

登录 [admin.lurus.cn](https://admin.lurus.cn) → 设置 → 安全 → 修改密码。

### Q: 忘记密码了怎么办？

在登录页面点击「忘记密码」，通过注册邮箱接收重置链接。

### Q: 可以删除账号吗？

可以。联系 [support@lurus.cn](mailto:support@lurus.cn) 申请账号注销。注销后所有数据（包括 API Key、鹿贝余额、交易记录）将被永久删除且无法恢复。

### Q: 支持哪些第三方登录？

目前支持 GitHub 和 Google OAuth 登录。可以在账号设置中绑定或解绑。

## 订阅与计费

### Q: 如何升级/降级套餐？

在 [admin.lurus.cn](https://admin.lurus.cn) → 订阅管理 → 「变更套餐」。

- **升级**: 立即生效，按比例补差价
- **降级**: 下个计费周期生效

### Q: 订阅到期后会怎样？

到期后自动降级为 Free 计划。你的 API Key 仍然有效，但受 Free 配额限制。所有数据保留，随时可以续费恢复。

### Q: 年付和月付有什么区别？

年付享受 8 折优惠（相当于免费用 2.4 个月）。年付期间不支持降级，但可以升级。

### Q: 团队计划中如何添加成员？

Team 及以上计划支持团队管理：
1. 管理后台 → 团队 → 邀请成员
2. 输入成员邮箱，发送邀请
3. 成员接受邀请后加入团队
4. 可为每个成员分配独立的 API Key 和配额

## 鹿贝

### Q: 鹿贝有什么用？

鹿贝（LB）是 Lurus 的通用积分货币，可用于：
- 支付超出订阅配额的 API 调用
- 订阅 Lucrum 付费策略
- 享受 VIP 折扣

### Q: 鹿贝会过期吗？

购买的鹿贝永久有效。活动赠送的鹿贝可能有有效期限制，以活动规则为准。

### Q: 鹿贝可以提现吗？

充值购买的鹿贝可以在未使用的情况下申请退款。Lucrum 策略收入结算的鹿贝可以提现到银行卡。

### Q: 如何查看鹿贝余额和消费记录？

登录管理后台 → 钱包，可以看到：
- 当前余额
- 收入明细（充值、奖励、策略收入）
- 支出明细（API 消费、策略订阅）

## 支付

### Q: 支持哪些支付方式？

| 方式 | 适用 |
|------|------|
| Stripe（信用卡/借记卡） | 全球用户 |
| Creem（加密货币） | 加密货币用户 |
| Epay（支付宝/微信） | 中国大陆用户 |

### Q: 付款后余额没到账？

支付通常在 1 分钟内确认。如果超过 5 分钟仍未到账：
1. 检查支付平台是否已扣款
2. 查看邮箱是否有支付确认邮件
3. 联系 [support@lurus.cn](mailto:support@lurus.cn) 并提供支付订单号

### Q: 如何申请发票？

管理后台 → 账单 → 申请发票。支持增值税普通发票和专用发票，通常 1 个工作日内发送到邮箱。

## 安全

### Q: 我的数据安全吗？

- 所有通信使用 HTTPS (TLS 1.3) 加密
- 密码使用 bcrypt 加密存储
- 支付由 PCI DSS 合规的第三方处理
- API 调用内容不会被存储（仅记录元数据用于计费）

### Q: API Key 被盗用了怎么办？

1. 立即登录控制台禁用该 Key
2. 创建新的 API Key
3. 检查调用日志，确认异常消费
4. 联系客服处理异常扣费
