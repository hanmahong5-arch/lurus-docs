# Webmail 快速开始

## 第一步：获取账号

Webmail 账号由管理员统一创建。如需开通，联系：

- **邮件**：admin@lurus.cn
- **说明**：提供姓名、部门，期望用户名（如 `zhangsan`）

账号创建后你会收到一封含初始密码的欢迎邮件。

## 第二步：登录 Web 界面

1. 打开浏览器，访问 [mail.lurus.cn](https://mail.lurus.cn)
2. 输入邮箱地址（如 `zhangsan@lurus.cn`）和密码
3. 点击「登录」

::: tip 修改初始密码
首次登录后请立即修改密码：点击右上角头像 → 「账号设置」→「修改密码」
:::

## 第三步：发送第一封邮件

1. 点击左上角「撰写」（或按 `C` 键）
2. 填写收件人地址
3. 输入主题和正文
4. 点击「发送」

### 发送带附件的邮件

1. 撰写邮件界面，点击附件图标（回形针）
2. 选择本地文件
3. 等待上传完成后点击「发送」

::: warning 附件大小限制
单封邮件附件总大小上限：**25 MB**
:::

## 第四步：配置邮件过滤规则（可选）

「设置」→「过滤规则」，可设置：

- 将特定发件人邮件自动归档到文件夹
- 自动转发到其他邮箱
- 自动回复（外出通知）

**示例：将 GitHub 通知归档**

```
条件：发件人包含 @github.com
动作：移动到文件夹「GitHub」
```

## 基本快捷键

| 快捷键 | 功能 |
|--------|------|
| `C` | 撰写新邮件 |
| `R` | 回复 |
| `A` | 全部回复 |
| `F` | 转发 |
| `Del` | 删除 |
| `S` | 搜索 |
| `?` | 显示所有快捷键 |

## 手机端访问

Webmail 支持移动浏览器直接访问，也可通过以下方式配置 App：

- **iOS**：使用系统邮件 App 或第三方 App（Canary Mail、Spark）
- **Android**：使用 Gmail App、K-9 Mail 或原生邮件 App

配置参数见 [客户端配置](/webmail/client-setup)。
