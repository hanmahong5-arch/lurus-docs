# 策略市场

Lucrum 策略市场是一个开放的量化策略生态，连接策略开发者和交易者。

---

## 对策略使用者

### 浏览与筛选

在策略市场中，你可以按多个维度筛选策略：

| 筛选条件 | 说明 |
|---------|------|
| 市场 | A 股、港股、美股、加密货币 |
| 策略类型 | 趋势跟踪、均值回归、套利、多因子 |
| 风险等级 | 保守、稳健、激进 |
| 最低资金 | 策略要求的最低投入资金 |
| 排序 | 收益率、夏普比率、最大回撤、订阅数 |

### 策略评估指标

每个上架策略都展示经过验证的量化指标：

| 指标 | 优秀标准 | 说明 |
|------|---------|------|
| **年化收益** | > 15% | 年化复合收益率 |
| **最大回撤** | < 20% | 历史最大亏损（峰谷差） |
| **夏普比率** | > 1.5 | 每单位风险对应的超额收益 |
| **Calmar 比率** | > 1.0 | 年化收益 / 最大回撤 |
| **胜率** | > 50% | 盈利交易占比 |
| **盈亏比** | > 1.5 | 平均盈利 / 平均亏损 |
| **运行天数** | > 90 天 | 策略实盘运行时长 |

::: tip 如何选策略？
不要只看收益率。一个夏普比率 2.0、最大回撤 10% 的策略，通常比年化 50% 但回撤 40% 的策略更适合大多数人。
:::

### 订阅与部署

1. 选择策略 → 查看详情页（含完整回测报告）
2. 点击「**订阅**」，确认费用
3. 进入「**我的策略**」→ 选择交易账户 → 设定资金分配
4. 点击「**启动**」，策略开始自动执行

### 费用说明

- 部分策略免费，部分按月订阅
- 订阅费从你的 [鹿贝钱包](/platform/billing#wallet) 扣除
- 交易产生的手续费由券商收取，与 Lucrum 无关

## 对策略开发者

### 上架流程

```
本地开发 → 回测验证 → 提交审核 → 上架市场 → 获取收入
```

1. **开发**: 在策略工作台编写策略代码
2. **回测**: 通过至少 2 年历史数据的回测验证
3. **提交**: 填写策略描述、风险等级、适用市场
4. **审核**: 平台审核策略合规性和风控（通常 1-3 个工作日）
5. **上架**: 审核通过后对所有用户可见

### 审核标准

| 项目 | 要求 |
|------|------|
| 回测数据量 | 至少覆盖 2 年历史数据 |
| 最大回撤 | 不超过 50%（超过需特别标注风险） |
| 风控措施 | 必须包含止损逻辑 |
| 代码质量 | 无内存泄漏、无无限循环风险 |
| 策略描述 | 完整说明策略逻辑、适用市场、风险提示 |

### 收益分成

策略产生的订阅收入按以下比例分配：

| 角色 | 分成比例 |
|------|---------|
| 策略作者 | **70%** |
| 平台 | **30%** |

收入以鹿贝形式结算到你的钱包，可提现到银行卡。

### 策略模板

Lucrum 提供多种策略模板，加速开发：

| 模板 | 适用场景 |
|------|---------|
| `TrendFollowing` | 趋势跟踪（均线交叉、突破策略） |
| `MeanReversion` | 均值回归（布林带、RSI 超买超卖） |
| `PairTrading` | 配对交易（统计套利） |
| `MultiFactorAlpha` | 多因子选股 |
| `GridTrading` | 网格交易（震荡市适用） |

## 策略类型说明

### 趋势跟踪

**原理**: 顺势而为，在趋势形成时入场，趋势结束时离场。

**典型信号**: 均线交叉、通道突破、动量指标

**适合**: 单边行情（牛市/熊市），不适合震荡市

### 均值回归

**原理**: 价格偏离均值后倾向于回归，在超跌时买入，超涨时卖出。

**典型信号**: RSI、布林带、Z-Score

**适合**: 震荡行情，不适合单边趋势

### 多因子选股

**原理**: 综合多个因子（估值、动量、质量、波动率等）对股票打分，买入高分股。

**典型因子**: PE/PB（估值）、ROE（盈利）、12 月动量、波动率

**适合**: 中长期持有，换仓频率低

### 配对交易

**原理**: 找到历史上高度相关的两只股票，当价差偏离均值时做空涨多的、做多跌多的。

**适合**: 追求低回撤、市场中性的稳健策略

## 风险提示

::: danger 投资有风险
- 历史回测表现不代表未来收益
- 量化策略可能在特定市场环境下失效
- 请根据自身风险承受能力合理配置资金
- Lucrum 不提供任何投资建议或收益保证
:::
