# GuShen 快速开始

10 分钟内完成第一个量化策略的创建和回测。

## 第一步：注册并登录

访问 [gushen.lurus.cn](https://gushen.lurus.cn)，使用邮箱或手机号注册。

::: tip 已有 Lurus 账号
GuShen 与 Lurus API 共享账号体系，直接登录即可，无需重新注册。
:::

---

## 第二步：创建策略

登录后点击「**策略管理**」→「**新建策略**」，选择创建方式：

### 方式 A — AI 生成（推荐新手）

在「策略描述」框中用自然语言描述你的逻辑：

```
当股价的 5 日均线上穿 20 日均线时，买入平安银行（000001.SZ）50% 仓位；
当 5 日均线下穿 20 日均线时，全部卖出。
```

点击「**AI 生成代码**」，几秒后自动生成可执行策略：

```python
def initialize(context):
    context.symbol = "000001.SZ"

def strategy(context):
    ma5  = context.ta.sma(symbol=context.symbol, period=5)
    ma20 = context.ta.sma(symbol=context.symbol, period=20)

    # Golden cross — buy signal
    if ma5[-1] > ma20[-1] and ma5[-2] <= ma20[-2]:
        context.order_percent(context.symbol, 0.5)

    # Death cross — sell signal
    elif ma5[-1] < ma20[-1] and ma5[-2] >= ma20[-2]:
        context.order_percent(context.symbol, 0)
```

### 方式 B — 手动编写

直接在代码编辑器中输入 Python 策略，参考 [策略编写指南](/gushen/strategy) 了解完整 API。

## 第三步：配置回测参数

| 参数 | 说明 | 建议值 |
|------|------|--------|
| 开始日期 | 回测起始时间 | 至少 3 年前（如 2020-01-01） |
| 结束日期 | 回测结束时间 | 近期日期，留 6 个月验证集 |
| 初始资金 | 模拟本金（元） | 1,000,000（便于计算收益率） |
| 手续费率 | 买卖费率 | 0.03%（A 股常见费率） |
| 滑点 | 成交偏差 | 0.1%（保守估计） |

::: tip 为什么要留验证集？
用 2020–2023 年数据优化参数，再用 2024 年数据验证，可以避免「在未来数据上表现好但实盘差」的过拟合。
:::

### 如何判断结果好坏？

| 指标 | 基准线 | 解读 |
|------|--------|------|
| **年化收益** | > 基准指数 | 高于沪深 300 说明策略有 alpha |
| **最大回撤** | < 20% | 超过 30% 则风险偏高，实盘心理压力大 |
| **夏普比率** | > 1.0 | 越高说明每单位风险获得的收益越高 |
| **胜率** | > 50% | 高频策略更依赖胜率；低频趋势策略胜率低无妨 |

## ⚠️ 回测陷阱：这些坑让新手亏损

::: danger 过拟合（最常见）
在历史数据上反复调整参数直到「看起来很好」，实盘却亏损。

**识别方法**：用你从未触碰的验证集测试——如果训练期年化 40%、验证期却只有 5%，说明过拟合严重。

**正确做法**：先定义策略逻辑，再选定固定参数，一次性回测，不要反复调参优化。
:::

::: warning 前向偏差（Look-ahead Bias）
策略代码中无意间使用了「当时还不知道的未来数据」。

**常见例子**：
```python
# ❌ 错误：用当天收盘价判断，但收盘才知道
if context.close[-1] > yesterday_close:
    context.order(...)   # 实际上下单时还没收盘

# ✅ 正确：用前一天数据判断
if context.close[-2] > context.close[-3]:
    context.order(...)
```
:::

::: warning 生存者偏差
只用「现在还存在的股票」做回测，会高估历史表现——已退市的烂股不在你的样本里。

GuShen 的 A 股数据集已包含退市标的，但使用第三方数据时请注意核实。
:::

## 常见问题

### 支持哪些市场？

| 市场 | 数据来源 | 最小频率 |
|------|---------|---------|
| A 股（沪深两市） | 同花顺 Level-1，前复权 | 分钟线 |
| 期货 | 交易所官方主力合约数据 | 分钟线 |
| 加密货币 | Binance 历史 K 线 | 分钟线 |

### 回测、模拟盘、实盘有什么区别？

| 模式 | 说明 | 资金 |
|------|------|------|
| **回测** | 历史数据验证，毫秒级完成 | 不涉及真实资金 |
| **模拟盘** | 实时行情模拟交易，验证策略延迟和滑点 | 不涉及真实资金 |
| **实盘跟踪** | 绑定券商账户自动下单 | 真实资金，开发中 |

::: warning
即使回测表现极好，也请先运行至少 1 个月模拟盘，再考虑实盘操作。历史不代表未来。
:::
