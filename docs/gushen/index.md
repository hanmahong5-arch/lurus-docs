# GuShen — AI 量化交易平台

## 什么是 GuShen？

**GuShen（谷神）** 是 Lurus 推出的 AI 驱动量化交易平台，将大语言模型的分析能力与量化交易策略深度结合。平台提供策略开发、回测验证、实盘执行的完整工作流，同时内置 AI 助手帮助你理解市场、优化策略。

> 名字取自《道德经》"谷神不死"——像山谷一样包容万象，持续适应。

---

## 核心能力

### AI 交易助手

基于 Lurus API 多模型能力的智能交易顾问，支持：

- **市场分析**: 实时解读 K 线形态、量价关系、技术指标
- **策略建议**: 根据当前持仓和市场情况给出操作建议
- **回测解读**: 用自然语言解释回测结果，指出策略弱点
- **风险预警**: 监控异常波动，及时提醒风险

AI 助手集成了 [MemX 记忆引擎](/memx/)，能记住你的交易偏好、历史决策和踩过的坑，随着使用越来越懂你。

### 策略市场

开放的量化策略生态，连接策略开发者和交易者：

| 角色 | 功能 |
|------|------|
| **策略作者** | 上传策略、设定价格、查看收益分成 |
| **策略使用者** | 浏览、订阅策略，一键部署到实盘 |

**收益分成**: 平台 30% / 策略作者 70%

### 配额与计费

GuShen 采用灵活的配额体系，确保资源合理分配：

```
计划限额 → Redis 月度计数 → 鹿贝余额兜底
```

| 层级 | 说明 |
|------|------|
| **计划限额** | 订阅套餐包含的月度 AI 调用次数 |
| **Redis 计数** | 实时追踪当前月已使用量 |
| **鹿贝兜底** | 配额用尽后，自动从 [鹿贝钱包](/platform/billing#wallet) 扣费（1 鹿贝 = 10,000 tokens） |

### 实时数据与执行

- **多市场接入**: A 股、港股、美股、加密货币行情
- **毫秒级延迟**: 策略信号到下单执行 < 100ms
- **风控引擎**: 仓位限制、止损止盈、最大回撤保护
