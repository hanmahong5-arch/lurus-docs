# Lucrum — AI 量化交易平台 <StatusBadge status="live" />

## 什么是 Lucrum？

**Lucrum** 是 Lurus 推出的 AI-Native 量化交易决策平台。核心理念：**自然语言是最好的编程语言**——用中文描述策略思路，AI 自动生成代码、执行<Term t="Backtest">回测</Term>、多维度评估。内置 11 个专业投资顾问 Agent（巴菲特/彼得林奇/利弗莫尔/西蒙斯等视角），全平台 Decimal.js 金融级精度计算（680+ 单元测试验证），零浮点误差。

> 名字源自拉丁语 "Lucrum"（收益），寓意精准洞察市场机会。

## 核心能力

### AI 策略生成与回测

- **自然语言 → 代码**: 中文描述策略意图，AI 自动生成 vnpy CtaTemplate 策略代码
- **金融级回测**: Decimal.js 全精度、A 股 100 <Term t="Round Lot">手</Term>整数倍约束、<Term t="Commission">佣金</Term>+<Term t="Stamp Duty">印花税</Term>+过户费+<Term t="Slippage">滑点</Term>
- **30+ 指标分析**: <Term t="Sharpe Ratio">夏普比率</Term>、<Term t="Max Drawdown">最大回撤</Term>、<Term t="Sortino">Sortino</Term>、<Term t="Calmar">Calmar</Term>、<Term t="Win Rate">胜率</Term>、盈亏比……
- **S/A/B/C/D 五级评分**: 4 维度加权（收益 30% + 风控 30% + 稳定性 25% + 效率 15%）

### 指标速览

| 指标 | 英文 | 含义 | 好的数值范围 |
|------|------|------|-------------|
| 夏普比率 | Sharpe Ratio | 每承受一单位风险获得的超额收益 | > 1.0 优秀, > 2.0 杰出 |
| 最大回撤 | Max Drawdown | 历史峰值到谷值的最大跌幅 | < 20% 可接受, < 10% 优秀 |
| 胜率 | Win Rate | 盈利交易占总交易的比例 | > 50% 基本, > 60% 良好 |
| 盈亏比 | Profit/Loss Ratio | 平均盈利 / 平均亏损 | > 1.5 可接受, > 2.0 优秀 |
| Sortino | Sortino Ratio | 类似夏普但只惩罚下行波动 | > 1.0 优秀 |
| Calmar | Calmar Ratio | 年化收益 / 最大回撤 | > 1.0 优秀 |

### 11 个 AI 投资顾问

基于 LangGraphJS 编排的多 Agent 投资分析系统：

| Agent | 投资风格 |
|-------|---------|
| **巴菲特** | 价值投资：寻找被低估的优质企业，长期持有 |
| **彼得·林奇** | 成长投资：发掘高增长潜力股，关注 PEG 指标 |
| **利弗莫尔** | 技术交易：跟随市场趋势，严格止损 |
| **西蒙斯** | 量化分析：数学模型驱动，统计套利 |
| **基本面分析师** | 财报解读，行业对比，估值建模 |
| **技术面分析师** | K 线形态、均线系统、量价关系 |
| **宏观分析师** | 货币政策、经济周期、资金流向 |
| **情绪分析师** | 舆情监控、市场情绪指标、散户行为 |
| **行业研究员** | 产业链调研，上下游分析，政策追踪 |
| **风控研究员** | 仓位管理、回撤控制、相关性分析 |
| **量化研究员** | 因子挖掘、回测验证、策略优化 |

- **多空辩论模式**: Bull vs Bear 对辩，避免单一视角偏见
- **记忆引擎集成**: 通过 [MemX](/memx/) 记住你的交易偏好和历史决策

### 策略市场

开放的量化策略生态，连接策略开发者和交易者：

| 角色 | 功能 |
|------|------|
| **策略作者** | 上传策略、设定价格、查看收益分成 |
| **策略使用者** | 浏览、订阅策略，一键部署到实盘 |

**收益分成**: 平台 30% / 策略作者 70%

### 配额与计费

Lucrum 采用灵活的配额体系，确保资源合理分配：

```
计划限额 → Redis 月度计数 → 鹿贝余额兜底
```

| 层级 | 说明 |
|------|------|
| **计划限额** | 订阅套餐包含的月度 AI 调用次数 |
| **Redis 计数** | 实时追踪当前月已使用量 |
| **鹿贝兜底** | 配额用尽后，自动从 [鹿贝钱包](/platform/billing#wallet) 扣费（1 鹿贝 = 10,000 tokens） |

### 实时数据与执行

- **多市场接入**: A 股、港股、美股、加密货币行情
- **毫秒级延迟**: 策略信号到下单执行 < 100ms
- **风控引擎**: 仓位限制、止损止盈、最大回撤保护
